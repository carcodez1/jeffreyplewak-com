- `Author:`

## Files

- `0001-ssot-pack.md`
- `0002-ui-polish.md`
- `manifest.json` (generated by signing script)

## Generate manifest + sign

```bash
bash scripts/prompts/sign-prompts.sh
```

## Verify

```bash
bash scripts/prompts/verify-prompts.sh
```

## Notes

- Signing keys are external to the repository.
- Do not commit key material.
